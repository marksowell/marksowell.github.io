# Contributing

Thanks for your interest in contributing.

This repository powers the public website at `https://marksowell.com` and is maintained as a personal site project.

## Before You Contribute

- For non-security bugs, broken links, or content issues, open a GitHub issue or pull request
- For security issues, do not open a public issue; follow [SECURITY.md](./SECURITY.md)
- For general support or contact guidance, see [SUPPORT.md](./SUPPORT.md)
- Project direction and final decisions are described in [GOVERNANCE.md](./GOVERNANCE.md)

## What Helps Most

Useful contributions include:

- Fixing broken links or incorrect content
- Improving accessibility or performance
- Fixing layout, rendering, or asset issues
- Small, well-scoped improvements to maintainability

## Contribution Guidelines

- Keep changes focused and easy to review
- Avoid large unrelated refactors
- Preserve the existing visual style unless the change is intentional and clearly justified
- Prefer simple solutions over added complexity
- Keep external runtime dependencies to a minimum

## Local Checks

Before opening a pull request, run:

- `pnpm lint`
- `pnpm run build`

## Pull Requests

When opening a pull request:

- Describe what changed and why
- Include screenshots for visible UI changes when helpful
- Mention any mobile or browser-specific behavior you tested

Not every contribution will be merged, but thoughtful improvements are appreciated.
