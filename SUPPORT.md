# Support

## Website Issues

For non-security issues with this repository or the public website, please open a GitHub issue with enough detail to reproduce the problem.

Examples:

- Broken links
- Rendering or layout issues
- Incorrect content
- Asset or page load problems

## Security Issues

Please do not report security issues in public GitHub issues.

Instead, see [SECURITY.md](./SECURITY.md) or email [mark@marksowell.com](mailto:mark@marksowell.com).

## General Contact

For professional or general contact, use [mark@marksowell.com](mailto:mark@marksowell.com).
